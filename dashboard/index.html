📖 How to get your keys

1
Discord Bot Token — Go to discord.com/developers → New Application → Bot → Reset Token. Enable Server Members Intent and Message Content Intent.
2
Server ID — In Discord, go to Settings → Advanced → Enable Developer Mode. Then right-click your server name → Copy Server ID.
3
Groq (Free) — Go to console.groq.com → Sign up free → API Keys → Create Key. No credit card needed.
4
Gemini (Free) — Go to aistudio.google.com → Get API Key → Create API key in new project. Free tier included.
5
After saving, run npm run start in your terminal. Then open Claude Desktop and say "Get my server info" to test.
