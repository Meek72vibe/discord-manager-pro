Privacy Policy

Discord Manager Pro · Last updated: March 2026 · Version 5.0
✅ ZERO DATA STORAGE

TL;DR: Discord Manager Pro is self-hosted software that runs entirely on your own machine. We (the developers) never receive, store, or have access to any of your data. Your Discord data stays between you, Discord, and the AI provider you choose.

1. Who We Are

Discord Manager Pro is an open-source MCP (Model Context Protocol) server available at github.com/meek72vibe/discord-manager-pro. It is free software maintained by independent developers.

2. What Data Is Accessed

When you run Discord Manager Pro, it accesses the following data from your Discord server via the Discord API:

  • Server information (name, member count, channel list, roles)
  • Channel messages (when using read or AI analysis tools)
  • Member information (usernames, roles, join dates)
  • Audit log entries
  • Server events, webhooks, and invites

3. How Data Is Processed

All data processing happens locally on your machine. The data flow is:

  • Your Discord server → Discord API → Your machine running Discord Manager Pro
  • For AI tools: Your machine → Your chosen AI provider's API (using your own API key) → Your machine
  • Results are returned to Claude Desktop on your machine

At no point does data pass through our servers — because we don't have any servers.

4. What We Store

We store absolutely nothing. Discord Manager Pro is stateless. The only local files created are:

  • config.json — Your API keys and settings, stored only on your machine
  • warnings.json — Member warning records, stored only on your machine

These files never leave your computer.

5. Third-Party Services

When you use AI-powered tools, message content is sent to your chosen AI provider using your own API key. Each provider has their own privacy policy:

  • Anthropic Claude: anthropic.com/privacy
  • Google Gemini: policies.google.com/privacy
  • Groq: groq.com/privacy-policy
  • Mistral AI: mistral.ai/privacy
  • Ollama (local): No data leaves your machine

You are responsible for reviewing the privacy policies of any AI provider you choose to use.

6. Discord API Usage

This bot uses Discord's official API and complies with Discord's Terms of Service and Developer Policy. We request only the minimum necessary permissions for the features you use. Privileged intents (GuildMembers, MessageContent) are used solely for the stated purpose of server management and AI analysis.

7. Children's Privacy

Discord Manager Pro is not directed at children under 13. Discord itself requires users to be at least 13 years old.

8. Your Rights

Since we store no data, there is nothing to delete or export. To remove all local data, simply delete the config.json and warnings.json files from your installation directory and remove the bot from your Discord server.

9. Changes to This Policy

We may update this privacy policy. Changes will be reflected in the GitHub repository with an updated date.

10. Contact

For privacy questions, open an issue at github.com/meek72vibe/discord-manager-pro/issues.
